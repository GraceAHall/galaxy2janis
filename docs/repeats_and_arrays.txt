

galaxy tool XML repeats 

- type: Param
- contents: list[Any]
- are nestable
    repeat: {
        param1,
        param2,
        {
            repeat: {
                param3
            }
        },
    }

galaxy workflow connection arrays

- example (multiqc)
    WORKFLOW STEP:
    "input_connections": {
        "results_0|software_cond|output_0|input": {
            "id": 3,
            "output_name": "text_file"
        },
        "results_0|software_cond|output_1|input": {
            "id": 4,
            "output_name": "text_file"
        }
    }
    TOOL XML:
    <repeat name="results">
        <conditional name="software_cond">
            <repeat name="output">
                <param name="input" />
            </repeat>
        </conditional>
    </repeat>
